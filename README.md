# PythonReactTaskTracker
## Описание проекта
Данный сервис предназначен для индивидуальной и коллективной работы с TODO-списками. Инструмент позволяет создавать и делиться списками дел. Обмен списками происходит через секретные ключи (UUID), которыми обладает каждый список.
### Функционал
1. Регистрация пользователя
2. Авторизация пользователя
3. Выход из аккаунта
4. Создание нового списка
5. Вход в существующий список через созданные списки
6. Вход в существующий список по его ключу
7. Копирование ключа списка
8. Создание задачи
9. Удаление задачи
10. Редактирование текста задачи
11. Редактирование описания задачи
12. Изменение статуса задачи 
## Запуск
1. `git clone https://github.com/thisisfunt/PythonReactTaskTracker.git`
2. `cd PythonReactTaskTracker`
3. `docker compose up`
